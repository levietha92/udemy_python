## CLI

- `pwd` = show where you are in path
- `ls` = list
- `cd` = change directory
- `mkdir` = make directory
- `touch` = create a file in directory
- `rm` = remove file
- `rm -rf` = remove folder # BE CAREFUL (will not be in bin)

https://github.com/appbrewery/terminal-mac-cheatsheet
